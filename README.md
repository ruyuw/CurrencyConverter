# Currency Converter

**Автор:** Смирнов Данил
**Курс:** Итоговая аттестация по GUI-разработке

## Описание
GUI-приложение для конвертации валют с использованием внешнего API, сохранением истории в JSON и загрузкой истории при запуске.

## Как получить API-ключ
1. Зарегистрироваться на [exchangerate-api.com](https://www.exchangerate-api.com)
2. Получить бесплатный ключ
3. Вставить ключ в переменную `API_KEY` в коде

## Запуск
```bash
pip install requests
python currency_converter.py
